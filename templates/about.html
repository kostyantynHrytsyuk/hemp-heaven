Welcome to Hemp Haven!


The best hemp market database for both agronomists and customers!

What framework does this website use?

Hemp Haven uses Flask for handling all the necessary functions of this web application

Flask is a micro web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components where pre-existing third-party libraries provide common functions.

What database management system does Hemp Haven use?

Hemp Haven uses PostgreSQL as a database management system.

PostgreSQL, also known as Postgres, is a free and open-source relational database management system emphasizing extensibility and SQL compliance. It was originally named POSTGRES, referring to its origins as a successor to the Ingres database developed at the University of California, Berkeley.
